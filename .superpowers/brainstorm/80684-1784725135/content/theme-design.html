配色方案对比

🌙 暗夜

#0f0f1a 背景
#1c1d21 卡片/Modal
#0d0e12 输入框
#4fd1c5 强调色
rgba(255,255,255,0.85) 主文字

☀️ 白昼

#f5f0eb 暖白背景
#ffffff 卡片/Modal
#f0ebe5 输入框
#0891b2 强调色（青蓝）
rgba(0,0,0,0.85) 主文字

切换方式

  • 设置菜单第三项改为「☀️ 切换皮肤」
  • 点击后弹出次级菜单：🌙 暗夜 / ☀️ 白昼
  • 选择后立即切换，设置存入 localStorage（选 dark 还是 light）
  • 使用 CSS 变量 + Ant Design algorithm: theme.darkAlgorithm | theme.defaultAlgorithm
  • 不需要 CSS 文件切换，所有颜色通过 CSS 自定义属性控制

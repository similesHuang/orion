关键变化

  • 去掉所有自定义 CSS 覆盖 — 不再设置 .ant-modal-header { background: transparent } 等覆盖。只用 className="settings-modal" + 主题 token
  • 用 Ant Design Form + Input — 替换自定义 FieldInput。用 Form.Item label="..." + Input /> 或 Input.Password
  • 输入框底色 — #0d0e12 比 Modal 底色再深一层
  • 整体实心 — Modal 背景 #1c1d21，所有子元素不透明。设置 Ant Design token colorBgElevated: '#1c1d21' 即全局生效
  • Modal 的 CSS 只需要 3 条规则 — 圆角 + 背景 + 输入框底色，其他都交给 Ant Design 主题 token

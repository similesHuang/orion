# Orion 主题系统 — 暗夜 GitHub Dark / 白昼中性灰

> 版本：v0.1
> 日期：2026-07-23

## 配色方案

### 🌙 暗夜（GitHub Dark）
| Token | 值 | 用途 |
|-------|-----|------|
| `--bg` | `#0d1117` | 主背景 |
| `--surface` | `#161b22` | 侧边栏/卡片 |
| `--elevated` | `#161b22` | Modal/弹出层 |
| `--input-bg` | `#0d1117` | 输入框 |
| `--ink` | `#e6edf3` | 主文字 |
| `--ink-dim` | `#c9d1d9` | 次要文字 |
| `--ink-mute` | `#8d96a0` | 弱文字 |
| `--line` | `#30363d` | 边框 |
| `--agent` | `#58a6ff` | 强调色(蓝) |
| `--you` | `#6366f1` | 用户(靛蓝) |
| `--ok` | `#3fb950` | 成功 |
| `--error` | `#f85149` | 错误 |
